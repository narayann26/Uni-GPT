The RAG Architecture

How raw university documents are transformed into instant, conversational intelligence.

1. Ingestion

Admin uploads official PDFs. System reads and chunks the text.

2. Vectorization

Chunks are converted to embeddings and stored in FAISS Vector DB.

3. Retrieval

User asks a query. System fetches only the most relevant document chunks.

4. Generation

LLM formats the retrieved data into a natural answer with citations.

System Capabilities

Strict Role-Based Access

Students see academic rules and PYQs. Faculty access grading schemas. Admins control the knowledge base. Complete data isolation.

Zero Hallucinations

By utilizing LangChain and strict prompt engineering, the AI is physically restricted from answering queries outside the uploaded university context.

PYQ Vault

Integrated Previous Year Question paper repository. Ask the AI to fetch "Data Structures PYQs from 2024" and get a direct download link.
